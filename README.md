# 🌦️ IntelliMeteo : A Nigerian Weather & Meteo Analytics Portal

## 🚀 Overview
A comprehensive weather intelligence platform designed for both public users and meteorological professionals in Nigeria.

The system provides real-time weather data, analytics, and specialized computational tools for aviation and agriculture.

---

## ✨ Key Features

- 🌍 Nationwide weather data visualization  
- 📊 Meteolytics dashboard (advanced analytics & charts)  
- ✈️ AeroMeteo tools (METAR/SPECI computations)  
- 🌱 AgroMeteo insights for agriculture  
- 🧠 Professional assessment module  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, Bootstrap, JavaScript  
- Backend: PHP  
- Database: MySQL  
- API: OpenWeather API
